Projects

VisioQuest: AI-Image-Inquiry

Sep 2023 - Jan 2024

This project presents VisioQuest: AI-Image-Inquiry, an interactive web application designed to answer questions about images using a multimodal Visual Question Answering (VQA) model. Trained on the DAQUAR dataset, which includes diverse indoor scenes, VisioQuest leverages advanced natural language processing and computer vision techniques through the integration of a RoBERTa text encoder and a ViT image encoder. Users can upload their own images and pose questions about the images to receive relevant answers.

Code is available on: [GitHub]

The Project is deployed on: [Hugging Face]

Tech Stack: Python, Pytorch, Transformers, NLTK, OpenCV, Streamlit

Image Caption Generator

Jan 2022 - May 2022

Developed a deep learning model for automatic image captioning, utilizing Convolutional Neural Networks (CNN) and Recurrent Neural Networks (RNN). The Xception model was employed to extract rich visual features from images, while the RNN was used to predict coherent sentences describing the image content. The model was trained and evaluated using the Flickr8k dataset, which contains a diverse collection of images paired with 5 descriptive captions.

Tech Stack: Python, TensorFlow, Keras, NLTK, NumPy, OpenCV
